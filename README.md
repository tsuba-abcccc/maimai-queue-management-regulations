# 项目简介

《舞萌机台排队通用管理办法》是一份面向舞萌玩家社区编写的非官方规则整理文本。

本项目以国家法律、行政法规以及规范性文件的表达形式为参考，将舞萌机台排队过程中常见的规则、习惯和争议进行系统整理，并建立统一的术语体系和逻辑体系。

项目本身具有一定的娱乐创作性质，通过“模拟管理办法”的形式，将原本依靠玩家默契维持的排队规则进行制度化表达。

同时，由于文本内容来源于玩家长期实践，因此其不仅具有创作观赏价值，也具有一定的现实参考意义。

---

# 为什么制作这个项目

舞萌机台排队长期依赖玩家之间的交流和默契。

在人较少时，玩家通常能够通过简单沟通解决顺序问题；但在人流增加、多机台环境、共同游玩、暂时离开以及发生争议时，仅依靠临场判断容易产生理解差异。

因此，本项目尝试回答：

* 谁应当优先取得游玩资格？
* “脸排”的顺序如何确认？
* 排卡时如何判断有效位置？
* 拼机需要满足什么条件？
* 暂离、放弃、退出如何区分？
* 发生争议时应当依据什么原则处理？

* 将玩家习惯转化为规范化条文；
* 对常见争议场景进行系统整理；
* 兼具娱乐阅读价值与现实参考价值；
* 持续接受玩家反馈并完善。

---

# 主要内容

本办法主要包括：

* 排队资格与游玩资格；
* 共同记忆排队（俗称“脸排”）；
* 登记排队（包括实体登记和电子登记）；
* 单人游玩与共同游玩；
* 排队顺序变化；
* 排队方式启用与转换；
* 多机台排队规则；
* 排队争议处理。

全文共十一章，当前版本共九十三条。

# 项目说明

本项目不创造官方规则，也不替代任何机厅管理制度。

实际排队过程中，应优先遵守：

* 舞萌官方相关规定；
* 机厅现场规则；
* 活动主办方安排；
* 玩家之间形成的合理约定。
