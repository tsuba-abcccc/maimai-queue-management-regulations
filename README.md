## 项目简介

《舞萌机台排队通用管理办法》是一份面向舞萌玩家社区编写的非官方规范文本。

因为舞萌排队规则只面向中国玩家，所以此项目全用中文编写。

本文以法律法规、公文的表达形式，对舞萌机台排队过程中常见的规则、习惯和争议进行系统整理，并建立统一的术语体系和逻辑体系，尽可能使不同场景能够按照一致的原则进行解释。

本文不创造新的规则，而是在尊重玩家社区长期形成的共识基础上，对已有实践进行规范化表达。

---

## 主要内容

本办法主要包括以下内容：

- 排队资格与游玩资格
- 共同记忆排队（俗称“脸排”）
- 标识登记排队（包括排卡和电子排卡）
- 单人游玩与共同游玩
- 排队顺序的演化
- 排队方式的启用与转换
- 多机台排队规则
- 排队争议处理

全文共十一章、一百零四条。
